# 司机台联动测试脚本

本目录包含独立的司机台联动测试脚本，用于测试仿真平台与司机驾驶模拟台之间的通信和联动控制。

## 协议概述

根据《轨交多系统平台接口协议汇总20260630.md》，司机台涉及以下通信协议：

| 协议 | 传输方式 | 服务器 | 端口 | 方向 |
|------|---------|--------|------|------|
| PLC协议 | TCP | 192.168.100.123 | 8001 | 上位机 ↔ PLC |
| 网络屏协议 | TCP | 192.168.100.122 | 8888 | 上位机 → 网络屏 |
| 信号屏协议 | TCP | 192.168.100.121 | 9999 | 上位机 → 信号屏 |
| 信号系统UDP | UDP | 动态 | 动态 | 信号系统 ↔ 总控数据库 |

## PLC 协议说明（文档 7.1 节）

PLC → 上位机报文为 **46 字节，大端序**，包含以下字段：

| 字节偏移 | 内容 | 说明 |
|---------|------|------|
| 0-3 | identify | 固定 0xAA55AA55 |
| 4-5 | total_len | 报文总长 46 |
| 6-7 | data_len | 数据区长度 22 |
| 8-19 | 时间 | 年/月/日/时/分/秒 (WORD) |
| 20-21 | verify_type | 校验类型 |
| 22-23 | verify_code | 校验值 |
| 24 | 指示灯标志 | 高断合/制动缓解不良/门关好/网络故障/AR具备 (BOOL) |
| 25 | 模式标志 | ATO具备/洗车模式/ATO激活/AR激活 (BOOL) |
| 26-27 | 车辆速度 | WORD，上位机传来的速度值 |
| 28 | 按钮标志 | 紧急制动/母线控制/强迫缓解/泵风/应急指挥/停放制动/电笛 (BOOL) |
| 29 | 门控标志 | 开/关左/右门 (BOOL) |
| 30-31 | 外部照明开关 | 0=停止 1=自动 2=近光 4=远光 |
| 32-33 | 门模式开关 | 0=半自动 1=手动 2=自动 |
| 34 | 按钮标志 | 高加速/司机室照明/模式升降确认/确认/AR/牵引复位/ATO启动 (BOOL) |
| 35 | 开关标志 | 洗车模式/钥匙开关/警惕/警惕允许解除 (BOOL) |
| 36-37 | 方向手柄 | 0=0位 1=向前 2=向后 |
| 38-39 | 主手柄 | 0=0位 1=牵引 2=制动 4=快制 |
| 40-41 | 牵引极位 | WORD，1% = 256（100% = 25600） |
| 42-43 | 制动极位 | WORD，同上 |
| 44-45 | 预留 | — |

## 使用方式

### 1. 启动 PLC 模拟器（纯软件，无需硬件）

```bash
# 后台运行，绑定 127.0.0.1
python -m driver_cab_test.plc_simulator --local
```

带交互控制台，可实时调参数：

```bash
python -m driver_cab_test.plc_simulator --local -i

PLC> speed 5000         # 设置车辆速度
PLC> hscb 1             # 高断合指示灯亮
PLC> ato_act 1          # ATO激活
PLC> dir 1              # 方向手柄向前
PLC> handle 1           # 主手柄牵引
PLC> traction 12800     # 牵引极位 50% (12800/256)
PLC> key 1              # 钥匙开关
PLC> status             # 显示当前全部状态
PLC> quit               # 退出
```

### 2. 监听 PLC 数据（连接模拟器）

```bash
# 持续接收，每秒刷新
python -m driver_cab_test.listen_plc --local
```

可选参数：
- `--count N` — 接收 N 包后退出（默认持续运行）
- `--interval N` — 刷新间隔秒数（默认 1.0）

```bash
python -m driver_cab_test.listen_plc --local --interval 2    # 每2秒刷新
python -m driver_cab_test.listen_plc --local --count 100     # 收100包退出
```

### 3. 连接真实硬件

不加 `--local` 默认连接真实 PLC（192.168.100.123:8001）：

```bash
python -m driver_cab_test.listen_plc
```
